#Golactic


Ligthweight GO Microservice Boilerplate

[**Explore the docs »**](https://github.com/TheYkk/golactic)


· [Report Bug](https://github.com/TheYkk/golactic/issues) · [Request Feature](https://github.com/TheYkk/golactic/issues)

### Built With

* Go
* zerolog
* OpenTelemetry
* gRPC
* Docker
* docker-compose
* Yaml config Support
* CLI 
* ENV Support
* GitHub Actions
